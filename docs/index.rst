scikit-pipes
============
Scikit-Learn useful pre-defined Pipelines Hub

#################################################################

This is repository of several Pipelines to use with Scikit-learn models in a direct way.
This package is still at an experimental stage.

Installation:
#############

Install scikit-pipes

It's advised to install sklearn-genetic using a virtual env, inside the env use::

   pip install scikit-pipes

.. |PythonMinVersion| replace:: 3.7
.. |ScikitLearnMinVersion| replace:: 0.21.3
.. |NumPyMinVersion| replace:: 1.14.5

scikit-pipes requires:

- Python (>= |PythonMinVersion|)
- scikit-learn (>= |ScikitLearnMinVersion|)
- NumPy (>= |NumPyMinVersion|)
